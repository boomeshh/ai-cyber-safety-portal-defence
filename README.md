# AI Cyber Safety Portal for Defence

AI-powered cyber incident reporting and safety portal for defence personnel, families, and veterans.

## Modules
- User Frontend
- Admin Frontend
- CERT Frontend
- Backend API

## Features
- Complaint submission
- Threat analysis
- Risk scoring
- Dashboard monitoring
- OCR and URL checking

## Tech Stack
- React.js
- FastAPI / Python
- SQLite / PostgreSQL
- OCR
- AI-based risk analysis

## Status
Prototype completed for project/demo purpose.
